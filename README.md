# 🏦 Bank Loan Data Analysis Dashboard

## 📌 Project Overview
This project analyzes bank loan application data to identify approval trends, default risk patterns, and customer financial behavior. The dashboard provides actionable insights to help financial institutions make data-driven lending decisions.

---

## 🎯 Business Objective
To analyze loan approval rates, customer demographics, and risk indicators in order to:
- Reduce default risk
- Improve loan approval strategy
- Identify high-risk customer segments
- Optimize lending performance

---

## 🛠 Tools & Technologies Used
- Power BI (Dashboard Development)
- Microsoft Excel (Data Cleaning & Preparation)
- SQL (SSMS – Data Extraction & KPI Queries)
- DAX (Calculated Measures & KPIs)

---

## 📂 Dataset Description
The dataset includes:
- Customer demographics (Age, Income, Employment Status)
- Loan amount & loan purpose
- Credit score
- Loan status (Approved / Rejected / Default)
- Debt-to-income ratio

---

## 📊 Key KPIs Developed
- Total Loan Applications
- Loan Approval Rate (%)
- Loan Default Rate (%)
- Average Loan Amount
- Income Distribution by Loan Status
- Credit Score vs Loan Approval Analysis

---

## 🔍 Key Insights
- Customers with lower income levels showed higher default probability.
- Higher credit scores significantly improved loan approval rates.
- Salaried employees had lower default rates compared to self-employed applicants.
- High loan amounts combined with high debt-to-income ratios increased default risk.

---

## 📈 Business Impact
This dashboard enables financial institutions to:
- Identify high-risk applicants before approval
- Improve loan approval decision accuracy
- Monitor default trends in real time
- Support risk management strategy
